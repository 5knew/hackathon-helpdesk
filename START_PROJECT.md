# Инструкция по запуску проекта

## Быстрый запуск

### Вариант 1: Использовать созданные скрипты

**Бэкенд:**
```powershell
.\start_backend.ps1
```

**Фронтенд (в отдельном терминале):**
```powershell
.\start_frontend.ps1
```

### Вариант 2: Ручной запуск

**Бэкенд:**
```powershell
cd backend
python main.py
```

**Фронтенд (в отдельном терминале):**
```powershell
cd frontend
npm run dev
```

### Вариант 3: С uvicorn (для бэкенда)

**Бэкенд:**
```powershell
cd backend
uvicorn main:app --reload --port 8002
```

## Проверка работы

После запуска проверьте:

1. **Бэкенд:** http://localhost:8002/docs
   - Должна открыться документация API
   - Должен быть виден эндпоинт `POST /tickets/{ticket_id}/feedback`

2. **Фронтенд:** http://localhost:3000 (или порт, указанный в консоли)
   - Должно открыться приложение

## Порты

- **Бэкенд:** 8002
- **Фронтенд:** 3000 (или другой, если 3000 занят)

## Остановка

Нажмите `Ctrl+C` в терминалах, где запущены процессы.

Или используйте:
```powershell
# Найти процесс на порту 8002
netstat -ano | findstr :8002

# Остановить процесс (замените PID на реальный)
taskkill /F /PID <PID>
```
