# Стратегия перевода датасета

## Проблема

Полный перевод всего датасета (28,586 строк) займет **~61 час** при текущей скорости (~7.7 сек/строка).

## Решения

### Вариант 1: Быстрая нормализация БЕЗ перевода ⚡ (РЕКОМЕНДУЕТСЯ)

Используйте `normalize_dataset_fast.py` для быстрой нормализации:

```bash
python normalize_dataset_fast.py
```

**Время:** ~5-10 минут  
**Что делает:**
- ✅ Удаляет дубликаты
- ✅ Нормализует метки
- ✅ Создает инструкции для разметчиков
- ❌ НЕ переводит (можно добавить переводы позже)

**Результат:** Нормализованный датасет готов для обучения

### Вариант 2: Перевод на выборке

Переведите только часть датасета для тестирования:

```bash
# Перевести 1000 строк (займет ~2 часа)
python normalize_and_translate_dataset.py --sample 1000
```

### Вариант 3: Перевод батчами (оптимизированная версия)

Используйте улучшенную версию с батчингом:

```bash
python normalize_and_translate_dataset.py
```

**Улучшения:**
- Собирает уникальные тексты (меньше переводов)
- Переводит батчами
- Использует кэш

**Время:** ~10-15 часов (вместо 61)

### Вариант 4: Использовать готовые переводы

Если у вас есть готовые переводы:
1. Нормализуйте датасет: `normalize_dataset_fast.py`
2. Добавьте переводы вручную или через другой инструмент
3. Объедините с нормализованным датасетом

## Рекомендация

**Для быстрого старта:**
1. Используйте `normalize_dataset_fast.py` - получите нормализованный датасет за 5-10 минут
2. Обучите модели на нормализованном датасете
3. Переводы можно добавить позже постепенно

**Для полного решения:**
1. Запустите оптимизированную версию на ночь
2. Или используйте облачный сервис перевода (Google Cloud Translation API, AWS Translate)
3. Или разбейте на части и переводите параллельно

## Сравнение вариантов

| Вариант | Время | Перевод | Готовность |
|---------|-------|---------|------------|
| normalize_dataset_fast.py | 5-10 мин | ❌ | ✅ Готов для обучения |
| Перевод 1000 строк | ~2 часа | ✅ | ✅ Тестовая версия |
| Оптимизированный перевод | ~10-15 часов | ✅ | ✅ Полный датасет |
| Полный перевод (текущий) | ~61 час | ✅ | ✅ Полный датасет |

## Вывод

**Для Hackathon:** Используйте `normalize_dataset_fast.py` - это даст нормализованный датасет за 5-10 минут, что достаточно для обучения моделей. Переводы можно добавить позже или использовать существующие переводы из датасета.
